DEMO -> http://www.datalogviewer.tech/

DataLogViewer is a Automtoive Sensor DataLog Visualizer built using Next.js with only native JavaScript, No Libraries

The app also uses a flask backend server to leverage machine learning for data interval classification
It is built with the following tools:
- TSFRESH (Time Series Feature Extraction)
- SciKit-Learn

I also plan to integrate nextAuth and allow users to make account and save datalogs.
